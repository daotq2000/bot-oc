================================================================================

  - Full width cards
  - Stacked sections
  - Touch-friendly buttons

  - 2 columns (md:grid-cols-2)
  - Compact spacing
  - Readable text

Desktop (> 1024px):
  - 3 columns (xl:grid-cols-3)
  - Generous spacing
  - Full-width form card

FORM VALIDATION
================================================================================

Required Fields:
  - Bot Name (min 3 characters)
  - Exchange (required)
  - Access Key (min 10 characters)
  - Secret Key (min 10 characters)
  - Future Balance Target (min $10)
  - Transfer Frequency (min 5 minutes)
  - Spot Transfer Threshold (min $5)
  - Spot Balance Threshold (min $10)
  - Withdrawal Address (required if auto-withdrawal enabled)

Optional Fields:
  - UID
  - Proxy
  - Telegram Chat ID

Error Display:
  - Red text below field
  - Clear error message
  - Field border turns red

Success:
  - Form closes
  - Bot list reappears
  - Success notification (if configured)
